# Print Shop Production System - Frontend

React frontend for the Print Shop Daily Production & Revenue System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- JWT-based authentication
- Role-based routing (Admin/Worker)
- Responsive design
- API integration with backend

## Tech Stack

- React 18
- React Router v6
- Axios for API calls
- Vite for build tooling
- date-fns for date formatting
